# Drivers-Positivo-Motion-Facilitado
Utilitário CLI em PowerShell feito por Robert Danilo (Suporte N3/ADS) para automatizar a instalação de drivers Positivo Motion. Garante instalação silenciosa e em lote via PnPUtil, com feedback visual de progresso. Reduza o tempo de setup pós-formatação e elimine erros manuais.
